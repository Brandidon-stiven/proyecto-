QUIENES SOMOS



Somos un sistema de informacion que le facilitara al agricultor el proceso de sembreado, control de inventario, compra y venta de insumos.





MODULOS

AGRICOLA

En este modulo el agricultor puede visulaizar distintos productos o insumos, seleccionarlos y ver el proceso de como cultivarlos.

INVENTARIO

En este modulo tiene acceso el administrador, el proveedor y el agricultor, los dos primero pueden agregar, actulizar o eliminar un insumo y el ultimo puede visualizar sus precios y ofertas, tambien poder descargar un reporte.

MANTENIMIENTO

Este modulo esta disponible para el administrador el tecnico y el agricultor, es esete modulo el agricultor puede asignar una cita a un tecnico para cualquier mantenimiento que necesite y el tecnico puede aceptar o rechazar la cita.

COMPRA Y VENTA

Esete modulo esta disponible para el administrador el agricultor y empresas aliadas aqui el productor puede subir cualquier insumo para que la empresa aliada lo compre y viceversa.
